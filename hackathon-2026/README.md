# AI Builders Hackathon 2026 – Bangladesh

Welcome to the official repository for **AI Builders Hackathon 2026 – Bangladesh**, a fully online, university-focused 7-day build challenge.

## Quick Overview

- **Mode:** Fully Online
- **Duration:** 7 Days
- **Team Size:** 1–4 university students
- **Prize Pool:** 3000 BDT total
  - 1st Place: 1500 BDT
  - 2nd Place: 1000 BDT
  - 3rd Place: 500 BDT
- **Recognition:** Digital certificates for top 10 teams

## Official Platforms

- **Registration:** Google Forms
- **Communication:** Discord
- **Submission:** GitHub repository link + Google Form
- **Judging:** Google Sheets
- **Cost Model:** Zero paid software required

## Submission Requirements (Summary)

Each team must submit:

1. Public GitHub repository link
2. `README.md` with problem, solution, setup, and demo details
3. 2–5 minute demo video link (unlisted accepted)
4. Team member list with university information
