# Resources

## Must-Use Platform
- z.ai docs and API playground (provided during kickoff)

## Beginner Path (Recommended)
1. Clone a starter kit from `starter-kits/`
2. Replace demo prompts and business logic
3. Build one clear social-good workflow
4. Add a short demo and README

## Project Building Checklist
- [ ] Problem statement is specific and socially relevant
- [ ] z.ai integration is implemented and tested
- [ ] README explains setup and usage
- [ ] Demo works in 2–5 minutes
- [ ] Repository is clean and understandable
