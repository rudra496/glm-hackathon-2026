## Free Development Tools

### Code & Environment
- [VS Code](https://code.visualstudio.com/) — Free code editor (recommended)
- [GitHub](https://github.com/) — Free hosting for your project repository
- [Google Colab](https://colab.research.google.com/) — Free Python notebooks in the browser

### Hosting & Deployment (Free)
- [GitHub Pages](https://pages.github.com/) — Free static website hosting
- [Vercel](https://vercel.com/) — Free web app deployment
- [Render](https://render.com/) — Free backend/API hosting

### APIs & AI Services (Free Tier)
- [Z.ai](https://z.ai) — Primary AI platform for this hackathon

- [OpenRouter](https://openrouter.ai/) — Free tier access to multiple LLMs
