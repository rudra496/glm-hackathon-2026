# Website (GitHub Pages)

This folder provides a modern landing page for the hackathon with animated visual elements and 3D-style interaction.

## Deploy
1. Push repository to GitHub.
2. Open repository **Settings → Pages**.
3. Choose **Deploy from branch**.
4. Branch: `main` (or default), Folder: `/website`.
5. Save.

## Customize Quickly
- Edit event text in `index.html`.
- Update colors and animation style in `style.css`.
- Adjust 3D interaction in `app.js`.
