# Registration Form Template

Use this structure in Google Forms / Microsoft Forms.

## Section 1: Basic Information
1. Full Name
2. Student ID
3. Department
4. Batch / Year
5. Email Address
6. Phone Number

## Section 2: Team Information
7. Team Name (if already formed)
8. Team Size (1/2/3/4)
9. Team Member Names + IDs
10. Need team matching support? (Yes/No)

## Section 3: Skill & Interest
11. Experience level (Beginner/Intermediate)
12. Preferred track (Education/Health/Environment/Accessibility/Other)
13. Have you used AI APIs before? (Yes/No)
14. Why do you want to join? (short answer)

## Section 4: Consent
15. Agree to code of conduct? (Yes/No)
16. Agree that project demo can be publicly shared? (Yes/No)
